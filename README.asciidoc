Server Sent Events chat application using Spring Webflux

start

[source,bash]
----
docker-compose up -d --remove-orphans
gradle bootRun
----

link:http://localhost:3000[test in browser]

[source,bash]
----
open http://localhost:3000
----

stop and cleanup

[source,bash]
----
gradle --stop
docker-compose down -v --remove-orphans
----
